└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐
│  STEP 3: Configure Settings                                   │
│                                                               │
│  Name:              chordypi-api                              │
│  Region:            Oregon (or closest to you)                │
│  Branch:            main                                      │
│  Root Directory:    server    ⚠️ CRITICAL!                   │
│  Environment:       Python 3                                  │
│  Build Command:     pip install -r requirements.txt          │
│  Start Command:     python app.py                            │
│  Instance Type:     Free                                      │
└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐
│  STEP 4: Add Environment Variables                            │
│  ➜ Open file: RENDER_ENV_VARS.txt                            │
│  ➜ Copy ALL variables                                         │
│  ➜ Paste into Render's Environment section                    │
│                                                               │
│  Required variables (7 total):                                │
│  ✓ FLASK_ENV                                                  │
│  ✓ FLASK_APP                                                  │
│  ✓ PORT                                                       │
│  ✓ PYTHON_VERSION                                             │
│  ✓ CORS_ORIGINS                                               │
│  ✓ SECRET_KEY                                                 │
│  ✓ JWT_SECRET_KEY                                             │
└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐
│  STEP 5: Deploy!                                              │
│  ➜ Click "Create Web Service" button                         │
│  ➜ Wait 3-5 minutes (watch build logs)                       │
│  ➜ Look for: ✅ "Your service is live"                       │
│  ➜ Copy your URL: https://chordypi-api.onrender.com          │
└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐

🆘 COMMON ISSUES:

❌ Build Failed?
   → Check: Root Directory is set to "server"
   → Check: requirements.txt exists in server folder

❌ Service Won't Start?
   → Check: All environment variables added
   → Check: PORT variable is set to 10000

❌ CORS Errors?
   → Check: CORS_ORIGINS includes https://chordypi.vercel.app
